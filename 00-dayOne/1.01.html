First Day Notes:

Notes:

  • When using Emmet, class in noted by a "." and id is noted by "#".
  • HTML stands for Hypertext Karkup Language.
  • DOCTYPE states the type of document. In this case, HTML.
  • our html tag denotes our language to be in English per lang="en".
  • The head tag houses metatdata, or detailed data about our doc.
  • UFT-8 specifies character encoding. Each character (letter, number, numberic value).
  • Viewport notes the device we are reading our document.
  • Title reflects the name and displays on the tab withing our browser.
  • A style tag can help us inject css into our document. *Not best practice.
  • Link allows us to connect to other files, such as our CSS (stylesheet).
  • The Body tag houses all the other tags that we want to display onto our browser. This is what the user sees.
